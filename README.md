LostLink — Intelligent Lost & Found Portal

**LostLink** is a centralized web platform developed as a Final Year Project (FYP).
It is designed to help university students and staff easily report, search, and match lost and found items within the campus, replacing unorganized manual notice boards and social media groups.

Technology Stack:
Our platform is built using the **MERN Stack** along with specialized cloud services for media handling and security.
**Frontend:** React.js & Tailwind CSS (For a modern, responsive UI)
**Backend:** Node.js & Express.js (To handle and process all user requests and data smoothly)
**Database:** MongoDB Atlas via Mongoose (For secure data storage)
**Cloud Storage:** Cloudinary API (For hosting and optimizing item images)
**Authentication:** JSON Web Tokens (JWT) & Bcrypt hashing
